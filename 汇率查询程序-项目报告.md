# 汇率查询 & 推送程序 — 项目报告

## 项目概述

纯 Python 3 命令行工具，从 **smbs.biz**（韩国）和 **chinamoney.com.cn**（中国外汇交易中心）获取当日汇率，支持**表格输出**、**JSON 输出**、**推送到外部 API**。全部使用 Python 标准库，**零第三方依赖**。

**版本**：1.0  
**开发日期**：2026-06-16  
**运行环境**：Python 3.6+（macOS / Linux / Windows）  
**代码行数**：417 行

## 程序架构

```
main()
  │
  ├─ 确定查询日期
  │    CLI --date 参数  >  config.json date 字段  >  当天
  │
  ├─ fetch_smbs(query_date)
  │    │  URL: http://www.smbs.biz/Flash/TodayExRate_flash.jsp?tr_date=YYYY-MM-DD
  │    │  编码: EUC-KR
  │    │  格式: URL query-string（?USD=1,510.00&updown1=1&diff1=-10.4&...）
  │    │  重试: 最多 5 次，间隔 60 秒
  │    └─ parse_smbs() → 57 种货币（外币 → KRW）
  │
  ├─ fetch_chinamoney(query_date)
  │    │  URL: https://www.chinamoney.com.cn/ags/ms/cm-u-bk-ccpr/CcprHisNew
  │    │  编码: UTF-8 / JSON
  │    │  格式: JSON（25 个货币对）
  │    │  重试: 最多 5 次，间隔 60 秒
  │    │  周末: 无数据时返回 None，不阻塞
  │    └─ parse_chinamoney_pair() → 25 条（外币 → CNY，CNY/X 自动翻转取倒数）
  │
  ├─ 错误处理
  │    ├─ smbs 失败 → 写日志到 logs/exrate_error_*.log，程序退出
  │    ├─ chinamoney 失败 → 提示跳过，继续运行
  │    └─ 推送失败 → 写日志到 logs/push_error_*.log
  │
  └─ 输出（三种模式）
       ├─ 表格 → python exrate.py
       ├─ JSON → python exrate.py --json
       └─ 推送 → python exrate.py --push
            └─ JSON URL 编码后拼到 config URL 尾部，GET 执行
```

## 数据源

### smbs.biz（韩元牌价）

| 项目 | 说明 |
|------|------|
| URL | `http://www.smbs.biz/Flash/TodayExRate_flash.jsp?tr_date=2026-06-16` |
| 编码 | EUC-KR（需解码） |
| 格式 | URL query string |
| 数据量 | **57 种货币** |
| 内容 | 外币兑韩元牌价，含涨跌值和涨跌方向 |
| 重试策略 | 5 次 × 60 秒间隔 |
| 失败处理 | 写日志 → 退出程序 |

### chinamoney.com.cn（人民币中间价）

| 项目 | 说明 |
|------|------|
| URL | `https://www.chinamoney.com.cn/ags/ms/cm-u-bk-ccpr/CcprHisNew` |
| 编码 | UTF-8 |
| 格式 | JSON |
| 数据量 | **25 个货币对** |
| 内容 | 中国外汇交易中心授权公布的人民币中间价 |
| 翻转规则 | `CNY/X` 类货币对自动翻转取倒数，统一为 **1 外币 = X CNY** |
| 重试策略 | 5 次 × 60 秒间隔 |
| 失败处理 | 非交易日无数据 → 跳过，不阻塞 |

## 输出格式

### 表格模式

```
  汇率  2026-06-16
-------------------------------------------------------
  韩元牌价 (smbs.biz) — 1外币 = ?韩元
-------------------------------------------------------
   源币 →    目标             汇率
-------------------------------------------------------
  USD →   KRW     1,510.0000
  JPY →   KRW       942.0400
  ...
-------------------------------------------------------
  共 57 种货币
-------------------------------------------------------
  人民币中间价 (chinamoney)
-------------------------------------------------------
   源币 →    目标             汇率
-------------------------------------------------------
  USD →   CNY         6.8108
  MOP →   CNY         0.8436
  ...
-------------------------------------------------------
  共 25 个货币对
```

### JSON 模式

```json
{
  "sources": {
    "smbs": {
      "description": "韩元牌价 (smbs.biz)",
      "date": "20260616",
      "rates": [
        { "from_currency": "USD", "to_currency": "KRW", "rate": 1510.0 }
      ]
    },
    "chinamoney": {
      "description": "人民币中间价 (中国外汇交易中心)",
      "date": "20260616",
      "rates": [
        { "from_currency": "USD", "to_currency": "CNY", "rate": 6.8108 }
      ]
    }
  }
}
```

### 推送模式

将上述 JSON 整体 URL 编码后，拼接到 `config.json` 配置的 URL 尾部：

```
http://your-api-server.com/api/exchange-rates?data=%7B%22sources%22%3A%20...
```

## 使用方式

```bash
# 表格输出（默认）
python exrate.py

# JSON 输出
python exrate.py --json

# 推送到外部 API
python exrate.py --push

# 指定日期
python exrate.py --date 2026-06-15
python exrate.py -d 2026-06-14 --json
python exrate.py -d 2026-06-14 --push

exrate.bat          # 表格
exrate_json.bat     # JSON
exrate_push.bat     # 推送
```

## 注意事项

1. **推送 URL 长度**：生成的 URI 约 13 KB，部分 Web 服务器默认限制 8 KB，需服务端配合调整
2. **推送协议**：目前仅支持 GET + URL 参数方式传递数据
3. **chinamoney 非交易日**：周日和法定节假日自动跳过，不影响主流程
4. **config.json**：必须为合法 JSON 格式，否则推送模式无法启动
5. **Windows Python**：首次运行需安装 Python 3 并勾选 "Add Python to PATH"
